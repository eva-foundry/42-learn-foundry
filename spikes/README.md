# spikes/

One-off exploratory scripts and notebooks.

## Naming Convention

`YYYYMMDD-<topic>.py` or `YYYYMMDD-<topic>.ipynb`

## Header template for each spike

```
# Spike: <topic>
# Date: YYYY-MM-DD
# Informs: <29-foundry path or "none">
# Conclusion: <one-line summary>
```

## Examples

- `20260221-prompty-load-test.py` -- tested Prompty.load() with policy-analysis.prompty
  Informs: 29-foundry/agents/azure_plane/app/conversation/conversation_agent.py
- `20260221-mcp-stdio-pattern.ipynb` -- explored MCP stdio transport pattern
  Informs: 29-foundry/mcp-servers/
